## ğŸ­ Animation Triggers

### On Page Load
1. **Navigation**: Slides down from top (1s delay)
2. **Hero Badge**: Fades in
3. **Hero Title**: Scales up from 0.9 with fade
4. **Hero Text**: Staggered fade from bottom
5. **3D Scene**: Particles spawn and start rotating

### On Scroll
1. **Workflow Section** (@ top of viewport)
   - Section pins in place
   - Progress bar fills
   - Steps appear one by one from right
   - Active step glows

2. **Services Section** (@ 80% viewport)
   - Header fades in
   - Cards appear with stagger (150ms)
   - Parallax movement on scroll

3. **Stats Section** (@ 70% viewport)
   - Cards fade up
   - Counters animate from 0 to target

4. **Pricing Section** (@ 85% viewport)
   - Cards appear with 3D rotation
   - Middle card elevated

### On Hover
1. **Navigation Links**: Color change to electric blue
2. **Service Cards**: Lift, glow, gradient overlay
3. **Pricing Cards**: Lift, scale, glow shadow
4. **Buttons**: Scale 1.05, glow increase
